Get started in 3 simple steps
1
Download a Model

Set Default Model
In Models → Model Manager, select the default model by clicking the radio button.

Start the Server
Click "Start Server" from the menubar, or enable auto-start in Settings. The server will load your default model automatically.

Metrics Reference
TTFT (Time to First Token)
Latency before the model starts responding. Measures prompt processing (prefill) speed. Lower is better.
TPOT (Time per Output Token)
Average time between each generated token. Measures decode speed. Lower is better.
tg TPS (Token Generation TPS)
Output tokens generated per second. Inverse of TPOT. Higher is better.
pp TPS (Prompt Processing TPS)
Input prompt tokens processed per second during prefill. Higher is better.
End-to-End Latency
Total time from request submission to complete response. Includes prefill + generation.
Total Throughput
Total tokens processed per second (input + output). Measures overall system utilization.
Batch Size
Number of concurrent requests processed simultaneously. Larger batches increase throughput but add per-request latency.
Speedup
Token generation throughput multiplier relative to single-request baseline (1x). Higher is better.
pp TPS/req (Per-request Prompt TPS)
Prompt processing throughput divided by batch size. Shows per-request prefill speed under batch load.
